# 🌸 Descripción del Proyecto

Este proyecto implementa una red neuronal artificial utilizando Python y Scikit-Learn para clasificar automáticamente flores del dataset Iris.

La inteligencia artificial aprende patrones a partir de características físicas como:

✅ Largo del sépalo  
✅ Ancho del sépalo  
✅ Largo del pétalo  
✅ Ancho del pétalo

Y clasifica flores en:

- Setosa
- Versicolor
- Virginica

# ⚙️ Funcionamiento del modelo

La red neuronal:

1. Carga el dataset Iris.
2. Divide datos en entrenamiento y prueba.
3. Entrena una red neuronal multicapa.
4. Aprende mediante épocas.
5. Calcula error y precisión.
6. Permite clasificación interactiva.

---

# 📊 Métricas utilizadas

Durante el entrenamiento se muestran:

✅ Error (Loss)  
✅ Precisión (Accuracy)

Cuando el error disminuye y la precisión aumenta:
➡️ significa que la red neuronal está aprendiendo correctamente.

---

# 🚀 Ejemplo de entrenamiento

```bash
Época: 1
Error: 1.02
Precisión: 57%

Época: 10
```
